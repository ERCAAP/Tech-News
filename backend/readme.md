# News Portal Backend API

Modern bir haber portalı için güçlü ve ölçeklenebilir backend API'si.

## 🚀 Özellikler

- TypeScript ile güçlü tip kontrolü
- JWT tabanlı kimlik doğrulama
- Role dayalı yetkilendirme (Admin/User)
- MongoDB ile veri yönetimi
- Haber yönetimi (CRUD işlemleri)
- Kullanıcı profil yönetimi
- Favori haber sistemi

## 🛠 Kurulum

1. Gereksinimleri yükleyin:
   ```bash
   npm install
   ```

2. `.env` dosyasını yapılandırın:
   - MongoDB bağlantı URL'sini ayarlayın
   - JWT secret key'i belirleyin
   - Diğer çevresel değişkenleri ayarlayın

3. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```
